## Installation

1. ```git clone git@github.com:ilyasilkin27/scarlet27-i3-dots.git ~/```
2. ```cd scarlet27-i3-dots```
3. Install dependncies using **yay** and **pacman**
4. Copy configs from **dot-config** to ~/.config
